## はじめに {#はじめに}

10年以上も風雨と雪にさらされたウッドデッキの **手すり** の傷みがひどくなってぼちぼち張り替えが必要になっています（図[1](#figure--our-cross-fence)）。

<a id="figure--our-cross-fence"></a>

{{< figure src="/images/our-deck-fence-katashina.jpg" caption="<span class=\"figure-number\">&#22259;1:  </span>現在のクロスフェンス。外見では分かりませんがかなり傷んでいます。" width="100%" >}}

手すりには多種多様なデザインがありますが、柵の方向によって基本的には **水平、垂直、クロス** の3種類に分けられます。工作の簡単さと強度、費用などいろいろ考えて結局 今と同じ **クロスフェンス** にすることにしました。

製作そのものは雪が融けるゴールデンウィーク以降になりますがまずは設計から。


## クロスフェンスの設計 {#クロスフェンスの設計}

図[2](#figure--fig-1)がフェンスユニットの設計図です[^fn_sketchup]。

<a id="figure--fig-1"></a>

{{< figure src="/images/deck-fence-unit.jpg" caption="<span class=\"figure-number\">&#22259;2:  </span>デッキフェンスユニットの寸法図。部品カテゴリごとに色分けしている。" width="100%" >}}

-   **<span style="color: yellow">手すり板</span>** は黄色
-   **<span style="color: purple">基礎板</span>** は紫色


### 主な仕様 {#主な仕様}

-   用材はすべて 2x4
-   クロスフェンスユニット（ **1,800(W) x 919mm(H)** ）は2区画で構成する。その1区画の **<span style="color: red">内法寸法</span>** は一辺 **843mm** の正方形
-   これを2ユニット連結してトータルの長さ＝ **3.6m** のフェンスとする。
-   クロス板は全8枚とも、長さ＝ **1,192mm** 、先端角度＝ **135°--90°--135°** で統一

[^fn_sketchup]: 3Dモデリングの[Sketchup Free](<https://app.sketchup.com/>)を使わせていただきました。


### フェンスユニット組み立て手順 {#フェンスユニット組み立て手順}


#### 1. フレームの組み立て {#1-dot-フレームの組み立て}

1.  フェンスのフレーム（図[3](#figure--fence-frame)）はコーススレッドの **垂直ビス止め** で組み立てる。

2.  **クロス板** （図[4](#figure--cross-board)）をはめ込み、ビス打ち固定（図[5](#figure--fig-4)）。

<a id="figure--cross-board"></a>

{{< figure src="/images/クロスフェンスのみ.jpg" caption="<span class=\"figure-number\">&#22259;4:  </span>クロス板の長さは 1,192mm" width="30%" >}}

<a id="figure--fig-4"></a>

{{< figure src="/images/クロス板をフレームに連結ーX.jpg" caption="<span class=\"figure-number\">&#22259;5:  </span>クロス板とフレームのビス止め法（X線モード）" width="60%" >}}


#### 2. 基礎板をデッキの土台にビス止め {#2-dot-基礎板をデッキの土台にビス止め}

-   デッキ土台に **基礎板** (1,800mm)を垂直ビス止めする。 **ブチルテープ** で防水する。


#### 3.フレームを基礎板に乗せてビス止め {#3-dot-フレームを基礎板に乗せてビス止め}

-   クロス板のはめ込みが終わったフレームを基礎板に乗せてビス止め。ビスはクロス板の **根元付近** のフレームに **斜め打ち** する。

{{< figure src="/images/coloredフレーム基礎板連結-2.jpg" caption="<span class=\"figure-number\">&#22259;6:  </span>フレームと基礎板の連結（（X線モード）" width="50%" >}}


#### 4. フレームに手すり板を乗せる {#4-dot-フレームに手すり板を乗せる}

1.  フレームの上に ****手すり板**** (1,800mm）を乗せて **木端(こば)** から斜めビス止め（図[7](#figure--handrail)）
2.  日の当たる時間の長い西側に2本、東側に3本の合計5本。ビス間間隔は300mm

## 今後の課題 {#今後の課題}

-   ユニット連結部から雨水がしみ込まないようにするためにシーリング材＋ブチルテープなどの使用を検討しています。

-   防腐・防虫のために、カット終了した部材の段階で水性クレオソート代替品系ペイントを六面すべてに塗布する予定です。
