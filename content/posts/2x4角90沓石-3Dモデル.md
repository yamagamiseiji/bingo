## 束石/沓石/デッキブロック {#束石-沓石-デッキブロック}

DIYでは、屋外の小屋や塀、棚などの支柱を支えるコンクリート製の ****基礎石**** をよく使います。色々な呼び方、種類、規格がありますが、参考資料にあげた図[11](#figure--fig-0)は ****2x4材**** （4x4も含む）向けの製品です。

****Sketchup free**** でDIYの図面を書くときには、この部材も書き込みたくなります。 ****3D Warehouse**** の中には海外の製品の3Dモデルはいろいろあるようです。しかし国内で販売されている製品のモデルはなかなか見つかりません。ならばと、自分で描くことにしました。

描き方はごく基礎的で単純なのですが、すぐに忘れる自分用の備忘として書いています。


## 3Dモデル作成手順 {#3dモデル作成手順}


#### 1. 沓石の躯体のもとになる直方体を描く {#1-dot-沓石の躯体のもとになる直方体を描く}

-   底面サイズ＝200x200mm、高さ＝250mmの直方体をプッシュ/プル(****P****)で書く。

<a id="figure--fig-1"></a>

{{< figure src="/images/沓石四角推台.png" caption="<span class=\"figure-number\">&#22259;1:  </span>躯体のもとになる直方体" width="50%" >}}


#### 2. 四角錐台の形に整形する {#2-dot-四角錐台の形に整形する}

-   天面の四辺の各辺を ****Shift**** を押しながら選択する。

<a id="figure--fig-2"></a>

{{< figure src="/images/沓石天面四辺選択.png" caption="<span class=\"figure-number\">&#22259;2:  </span>天面の四辺を選択" width="50%" >}}


#### 3. 天面を180x180に縮める {#3-dot-天面を180x180に縮める}

-   スケールツール(****S****)を選び、 ****Ctrlキー**** を押しながら ****角**** （コーナー）を内側にドラッグすることで、天面が中心に向かって均等に縮小される。
-   縮小率は 0.9、数値は数字入力窓に入力

<a id="figure--fig-3"></a>

{{< figure src="/images/沓石S縮小.png" caption="<span class=\"figure-number\">&#22259;3:  </span>Ctrl+S で天面を縮める" width="50%" >}}


#### 4. 天面中央に4x4用の穴を開ける {#4-dot-天面中央に4x4用の穴を開ける}

-   直線ツール(****L****)で天面中央に95x95mmの正方形を描く。

<a id="orgf2b7749"></a>

o<img src="/images/沓石ケガキ.png" alt="沓石ケガキ.png" width="50%" />


#### 5. 4x4の穴あけ {#5-dot-4x4の穴あけ}

-   95x95mmの正方形を選択し、プッシュ/プル(****P****)で深さ80mm の穴をあける。

<a id="figure--fig-5"></a>

{{< figure src="/images/沓石4x4穴.png" caption="<span class=\"figure-number\">&#22259;5:  </span>深さ80mmの4x4用穴あけ" width="50%" >}}


#### 6. 溝の十字型を作る {#6-dot-溝の十字型を作る}

-   十字溝の深さ40mmに対して深めの60mm、長さ180mmに対して長めの500mm、幅40mmに対して40mmの棒を2本作り、十字に組んで下図のような十字型とする[^fn_color]。
-   中心部のガイドポイントに注目[^fn_flip]
-   グループ作成すること

<a id="figure--fig-6"></a>

{{< figure src="/images/沓石十字型.png" caption="<span class=\"figure-number\">&#22259;6:  </span>溝用の十字型" width="50%" >}}

[^fn_color]: 十字型は見やすさのために色付けした。

[^fn_flip]: マークしたガイドポイントが底面になるように十字型を上下ひっくり返す必要があるが、それには反転ツール(****flip****)を使う。


#### 7. 四角錐台にガイドポイントを打つ {#7-dot-四角錐台にガイドポイントを打つ}

-   オブジェクトをX線透視モードにする。
-   四角錐台と十字型を合体させる時に、十字型のガイドポイントが来る位置（天面レベルから40mm下方で躯体中心線上の位置）にガイドポイントを打つ[^fn_guideP]。

<a id="figure--fig-7"></a>

{{< figure src="/images/沓石ガイドP.png" caption="<span class=\"figure-number\">&#22259;7:  </span>躯体中心線上のガイドポイントに注目" width="50%" >}}

[^fn_guideP]: ガイドライン上にガイドポイントを打つには、 ****T**** しつつ ****Ctrl**** を押す。


#### 8. 十字型と四角錐台の合体 {#8-dot-十字型と四角錐台の合体}

-   オブジェクトをX線透視モードにする。
-   十字型底面の中央に打ったガイドポイントをつまんで移動ツール(****M****)で移動させ、躯体中心線上のガイドポイントに乗せる。

<a id="figure--fig-8"></a>

{{< figure src="/images/沓石合体-X.png" caption="<span class=\"figure-number\">&#22259;8:  </span>十字型と四角錐台の合体" width="50%" >}}


#### 9. 面を交差 {#9-dot-面を交差}

-   合体後、全オブジェクトを選択、右クリックで表示されるメニューから、「面を交差」➡ 「選択範囲と交差」を選ぶ[^fn_intersec]。
-   四角錐台の斜面と十字型がぶつかっている部分（面の交差している箇所）に「新しい線」が自動的に書き込まれる。

<a id="figure--fig-9"></a>

{{< figure src="/images/沓石intersection.png" caption="<span class=\"figure-number\">&#22259;9:  </span>面の交差、選択範囲の交差の終了後" width="50%" >}}

[^fn_intersec]: 境界に線を引くことで不要になったオブジェクトを削除するため。


#### 10. 完成 {#10-dot-完成}

-   不要な十字型の面や線などを削除[^fn_not-only]してできあがり[^fn_shuft-E]。
-   モデルをSketchup 3D Warehouse に登録して必要な時にダウンロードして使う。

<a id="figure--fig-10"></a>

{{< figure src="/images/沓石完成品.png" caption="<span class=\"figure-number\">&#22259;10:  </span>完成した2x4沓石90角の3Dモデル" width="50%" >}}

[^fn_not-only]: 削除だけでなく、場合によっては線を描き加えることが必要になる場合もある。

[^fn_shuft-E]: 多数の線分や面を削除するときには、加算選択（ ****Shift**** キーを押しながらクリック）が有効。
